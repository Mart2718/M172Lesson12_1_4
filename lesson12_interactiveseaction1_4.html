Section 1: Graphing Sine and Cosine

Source: OpenStax Algebra and Trigonometry 2e — Section 8.1

Definition

Sinusoidal Function: A function with the same general shape as a sine or cosine function.

General forms:

\[ y = A\sin(Bx - C) + D \qquad \text{or} \qquad y = A\cos(Bx - C) + D \]

Period: \(2\pi\)  |  Domain: \((-\infty, \infty)\)  |  Range: \([-1, 1]\)

\(y = \sin(x)\) is ODD — symmetric about the origin  [\(\sin(-x) = -\sin(x)\)]

\(y = \cos(x)\) is EVEN — symmetric about the \(y\)-axis  [\(\cos(-x) = \cos(x)\)]

⚠ Key Reminder: The outputs of sine and cosine are coordinates on the unit circle — always between \(-1\) and \(1\). They are periodic because the unit circle repeats every full rotation (\(2\pi\) radians).

🤔 Think First — Before We Graph

Imagine tracing the y-coordinate of a point moving around the unit circle. What shape do you predict the graph of \(y = \sin(x)\) will make? Why?


✅ The sine graph is a wave! As the angle \(x\) increases from \(0\) to \(2\pi\), the \(y\)-coordinate on the unit circle rises from \(0\) to \(1\), falls back through \(0\) to \(-1\), and returns to \(0\) — creating one full cycle of the characteristic S-shaped wave.

Zeros: \(x = k\pi\), \(k \in \mathbb{Z}\)
Graph of \(f(x) = \cos(x)\)

The cosine function starts at its maximum when \(x = 0\). Compare it to the sine function — how are they related?

🔑 Sine vs. Cosine Connection: \(\cos(x) = \sin\!\left(x + \dfrac{\pi}{2}\right)\) — cosine is just sine shifted left by \(\dfrac{\pi}{2}\). They are the same wave, just out of phase!

Section 2: Graphing the Tangent Function

Source: OpenStax Algebra and Trigonometry 2e — Section 8.2

Definition

Tangent Function: \(\displaystyle\tan(x) = \frac{\sin(x)}{\cos(x)}\) — an ODD function with period \(\pi\).

Features of \(y = A\tan(Bx)\):

  • Stretching/compressing factor: \(|A|\)
  • Period: \(\dfrac{\pi}{|B|}\)
  • Vertical asymptotes wherever \(\cos(x) = 0\), i.e., at \(x = \dfrac{\pi}{2} + k\pi\) (\(k\) any integer)
  • Domain: all real numbers except asymptote locations
  • Range: \((-\infty, \infty)\)
  • No amplitude — the function grows without bound near each asymptote

⚠ Key Reminder: Tangent is undefined wherever cosine equals zero — those are the vertical asymptote locations. The graph passes through the origin and increases through each period. Each period looks like an S-curve between two asymptotes.

🤔 Think First

If \(\tan(x) = \frac{\sin(x)}{\cos(x)}\), what happens to the output when \(\cos(x)\) gets very close to zero? What should the graph look like near those \(x\)-values?


✅ Vertical asymptotes! When \(\cos(x) \to 0\), the denominator approaches zero while the numerator \(\sin(x)\) stays near \(\pm 1\). So the ratio \(\frac{\sin(x)}{\cos(x)}\) explodes toward \(\pm\infty\) — creating vertical asymptotes at every \(x = \frac{\pi}{2} + k\pi\).
Graph of \(f(x) = \tan(x)\)

Observe the asymptotes (dashed red) and the increasing behavior within each period. Notice that the graph has no amplitude.

🔑 Why no amplitude? Amplitude measures the maximum distance from the midline. Since \(\tan(x)\) grows without bound near its asymptotes, there is no finite maximum — so amplitude is not defined for tangent (or cotangent).

Section 3: Graphing Reciprocal Trigonometric Functions

Source: OpenStax Algebra and Trigonometry 2e — Section 8.2

Three additional trig functions are the reciprocals of sine, cosine, and tangent. Wherever the parent function equals zero, the reciprocal has a vertical asymptote (you can't divide by zero).

Cosecant

\(\displaystyle\csc(x) = \frac{1}{\sin(x)}\)

Vertical asymptotes: \(x = k\pi\)  |  Domain: all reals except \(x = k\pi\)

Range: \((-\infty, -1] \cup [1, \infty)\)  |  Period: \(2\pi\)

Secant

\(\displaystyle\sec(x) = \frac{1}{\cos(x)}\)

Vertical asymptotes: \(x = \dfrac{\pi}{2} + k\pi\)  |  Domain: all reals except \(x = \dfrac{\pi}{2} + k\pi\)

Range: \((-\infty, -1] \cup [1, \infty)\)  |  Period: \(2\pi\)

Cotangent

\(\displaystyle\cot(x) = \frac{\cos(x)}{\sin(x)} = \frac{1}{\tan(x)}\)

Vertical asymptotes: \(x = k\pi\)  |  Domain: all reals except \(x = k\pi\)

Range: \((-\infty, \infty)\)  |  Period: \(\pi\)   (graph decreases through each period)

Quick Reference — Reciprocal Trig Functions
FunctionDefined AsPeriodAsymptotes AtRange
\(\csc(x)\)\(1/\sin(x)\)\(2\pi\)\(x = k\pi\)\((-\infty,-1]\cup[1,\infty)\)
\(\sec(x)\)\(1/\cos(x)\)\(2\pi\)\(x = \frac{\pi}{2}+k\pi\)\((-\infty,-1]\cup[1,\infty)\)
\(\cot(x)\)\(\cos(x)/\sin(x)\)\(\pi\)\(x = k\pi\)\((-\infty,\infty)\)

🤔 Cot vs. Tan — Key Differences

Before revealing, write one difference between \(\tan(x)\) and \(\cot(x)\) that you noticed in the Desmos graph:


✅ Three key differences between \(\cot(x)\) and \(\tan(x)\):
  1. Direction: \(\tan(x)\) increases through each period; \(\cot(x)\) decreases
  2. Asymptotes: \(\tan(x)\) has asymptotes at \(x = \frac{\pi}{2} + k\pi\); \(\cot(x)\) has asymptotes at \(x = k\pi\)
  3. Period: Both have period \(\pi\), but they are shifted — \(\cot(x)\) starts from a high value on the left and falls to the right

Section 4: Describing Transformations of Trigonometric Functions

Source: OpenStax Algebra and Trigonometry 2e — Section 8.1

Transformed Sinusoidal Form

\[ f(x) = A\sin\!\bigl(B(x - D)\bigr) + C \]

ParameterNameEffect
\(A\)AmplitudeVertical stretch/compress; amplitude = \(|A|\). If \(A < 0\), reflection over \(x\)-axis.
\(B\)Period factorPeriod = \(\dfrac{2\pi}{|B|}\). If \(|B|>1\) → compress; if \(|B|<1\) → stretch.
\(D\)Phase (horizontal) shiftShift right by \(D\) (left if \(D < 0\)). Factor out \(B\) first!
\(C\)Vertical shiftMidline moves to \(y = C\); range becomes \([C - |A|,\; C + |A|]\).

⚠ Key Reminder: The horizontal shift is \(D\) (not \(BD\)). You must factor out \(B\) first so the form is \(B(x - D)\), then read \(D\) directly. Example: \(y = \sin(3x - \pi)\) → factor: \(y = \sin\!\bigl(3\!\left(x - \tfrac{\pi}{3}\right)\bigr)\) → phase shift = \(\tfrac{\pi}{3}\), NOT \(\pi\).

🤔 Predict the Effect

For \(g(x) = 1.5\cos(2x)\) compared to \(f(x) = \cos(x)\): what do you predict will change in the graph? What stays the same?


✅ Predicted effects:
  • \(A = 1.5\) → amplitude increases from \(1\) to \(1.5\) (taller waves)
  • \(B = 2\) → period = \(\frac{2\pi}{2} = \pi\) (shorter, faster cycles — twice as many waves in the same window)
  • No phase shift (\(D = 0\)) and no vertical shift (\(C = 0\)) — same starting point and same midline

Worked Example — Describing a Full Transformation

Example: \(y = 3\sin\!\left(\dfrac{3}{2}(x - \pi)\right) - 4\)

Try to identify each transformation before clicking to reveal each step.

Task: Describe all transformations and identify the domain and range.

Step 1 — Identify the form and parameters

The equation is in the form \(y = A\sin\!\bigl(B(x - D)\bigr) + C\) with:

\(A = 3\),  \(B = \dfrac{3}{2}\),  \(D = \pi\),  \(C = -4\)

Step 2 — Amplitude

Amplitude \(= |A| = |3| = \mathbf{3}\)

The graph reaches 3 units above and 3 units below the midline.

Step 3 — Period

\(\text{Period} = \dfrac{2\pi}{|B|} = \dfrac{2\pi}{\;\frac{3}{2}\;} = 2\pi \cdot \dfrac{2}{3} = \dfrac{4\pi}{3}\)

Step 4 — Horizontal (Phase) Shift

\(D = \pi\) → shift right by \(\pi\) units

Remember: the equation is already factored as \(B(x - D)\), so read \(D\) directly.

Step 5 — Vertical Shift & Midline

\(C = -4\) → shift down 4 units. The midline is \(y = -4\).

Step 6 — Domain and Range

Domain: \((-\infty, \infty)\) — sine is defined for all real numbers

Range: \([C - |A|,\; C + |A|] = [-4 - 3,\; -4 + 3] = \mathbf{[-7,\; -1]}\)

Summary:
  • Amplitude: \(3\)  |  Period: \(\dfrac{4\pi}{3}\)  |  H-shift: right \(\pi\)  |  V-shift: down \(4\)
  • Domain: \((-\infty, \infty)\)  |  Range: \([-7, -1]\)

Model Answer:
